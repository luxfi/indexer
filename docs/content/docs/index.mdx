# Lux Indexer

A Go-based unified indexer for all Lux Network chains. Designed as a complete replacement for Blockscout's Elixir backend, providing 100% feature parity with improved performance and maintainability.

## Features

- **Multi-Chain Support**: Index C-Chain, P-Chain, X-Chain, Zoo, and custom chains
- **Full Blockscout API Compatibility**: Drop-in replacement with REST, GraphQL, and WebSocket APIs
- **High Performance**: 430K+ ops/sec for indexing operations
- **Native DeFi Indexing**: Built-in support for AMM, DEX, lending, and bridge protocols
- **Multiple Storage Backends**: PostgreSQL, SQLite, BadgerDB, MongoDB, Dgraph

## Quick Start

```bash
# Install
go install github.com/luxfi/indexer/cmd/indexer@latest

# Run against local node
indexer -chain evm -rpc http://localhost:9650/ext/bc/C/rpc

# Run against mainnet
indexer -chain evm -rpc https://api.lux.network/ext/bc/C/rpc
```

## Chain Support

| Chain | Type | Description |
|-------|------|-------------|
| C-Chain | EVM | Ethereum-compatible smart contracts |
| P-Chain | Platform | Staking and validator management |
| X-Chain | DAG | Asset exchange with UTXO model |
| Zoo | EVM | Zoo network EVM chain |

## Architecture

```
indexer/
├── evm/          # Unified EVM indexer (C-Chain, Zoo)
├── pchain/       # P-Chain (Platform) indexer
├── xchain/       # X-Chain (Exchange) indexer
├── storage/      # PostgreSQL, SQLite, BadgerDB backends
├── e2e/          # End-to-end tests
└── docs/         # Documentation site
```
